# VoiceRecognition
A speech-to-text app that converts spoken words into text using the device's recognition engine.

## Features
- Real-time speech recognition  
- Error handling messages  
- Clean text display  

## How to Run
1. Import into MIT App Inventor.
2. Connect device/emulator.
3. Run with **AI Companion**.

## Tech Stack
- MIT App Inventor  
- Speech Recognizer component
